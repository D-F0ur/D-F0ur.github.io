环境搭建

在Sublime Text中安装Markdown语法高亮主题

在Package安装Monokai Extended主题和Markdown Extended

在Sublime Text中安装Evernote插件

在Package Control中搜索Evernote安装，根据提示操作，并打开https://app.yinxiang.com/api/DeveloperToken.action, 输入Developer Token和NoteStore URL，连接你的印象笔记账户。

在Sublime Text中安装Markdown Preview插件

Markdown非所见所得模式，所以我们可以借助Markdown Preview这款插件来满足写完文档后预览的需要。在浏览器中选择使用Github风格来预览，排版很漂亮。同样，在Package Control中搜索安装。

工作流程

  1. 在Sublime Text中新建一篇Markdown文档
  2. 在无干扰模式下完成文章
  3. 在浏览器中使用Github风格预览，修改排版
  4. 将文章保存到Evernote中，完成知识的存档
